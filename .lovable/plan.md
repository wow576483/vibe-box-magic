

# Khodari – Fresh Produce Box Delivery Landing Page

A high-converting, mobile-first, RTL Arabic landing page for a fresh-produce box delivery service in Constantine, Algeria.

## Brand & Design System
- **Primary color**: `#78B05F` (fresh green) with light neutrals and dark charcoal for text
- **Style**: Warm, trustworthy, modern — rounded cards, soft shadows, subtle gradients
- **Font**: Friendly sans-serif (e.g. Cairo for Arabic support)
- **Direction**: Full RTL support throughout
- **Animations**: Scroll-triggered fade-ins, hover effects, micro-interactions — tasteful and performant

---

## Page Sections

### 1. Hero – Problem + AIDA
- Bold headline about the pain of market trips for busy moms in Constantine
- 2–3 bullet benefits, primary CTA button ("اطلبي سلتك الآن")
- Reassurance text: cash on delivery, Constantine delivery
- Illustration area (placeholder image of mom receiving veggie box)
- **Animations**: Fade/slide-in on load, CTA hover scale effect

### 2. Problem Section – "يومك مع السوق"
- 3–4 problem cards (time wasted, fatigue, stress with kids, quality disappointment)
- Icon + short description per card
- **Animations**: Staggered upward fade-in on scroll, hover lift + shadow

### 3. Before/After Strip (BAB)
- Two-column layout: "قبل خضاري" vs "بعد خضاري"
- Bridge sentence in the center
- Diagonal gradient from desaturated to green
- **Animations**: Gradient slide transition on scroll, hover brightness

### 4. Solution Intro – "خضاري يحوّل السوق لتجربة مريحة"
- Title + paragraph introducing Khodari as the solution
- 3–4 benefit cards with icons (time saving, guaranteed quality, clear pricing, WhatsApp ordering)
- **Animations**: Bottom fade-in with stagger, icon "breathe" pulse

### 5. How It Works – 3 Steps
- Horizontal timeline / numbered cards: Choose box → We prepare & confirm → We deliver, pay on receipt
- Steps connected with visual line/arrows
- **Animations**: Sequential highlight on scroll, hover raise effect

### 6. Boxes / Pricing Section
- 3 pricing-style cards (Basic box with "most popular" badge, Family box, Couples/Students box)
- Each card: name, target audience, duration, example contents, price, CTA button
- Popular box highlighted with green accent
- **Animations**: Hover scale + shadow, button hover states

### 7. Social Proof – Testimonials
- 2–3 testimonial cards with name, neighborhood, quote, star rating
- **Animations**: Fade-in on scroll, hover tilt/depth effect

### 8. FAQ Accordion
- 5–7 common questions (delivery areas, timing, payment, quality, customization)
- Smooth accordion expand/collapse with rotating chevron icons
- Light background with subtle green accents

### 9. Final CTA Section
- Recap sentence (problem → solution)
- Large CTA button + secondary WhatsApp link
- Soft green gradient background
- **Animations**: Scroll-in, clear hover/press states

---

## Technical Approach
- Fully responsive mobile-first layout scaling to desktop
- RTL direction throughout with proper Arabic typography
- Scroll-based animations using IntersectionObserver
- Smooth scrolling for in-page anchor navigation
- Clean, component-based React architecture for easy future updates
- All copy uses placeholder Arabic text, ready for final replacement
